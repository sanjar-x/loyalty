# Иерархия территориального деления России

## Полная цепочка от международного стандарта до конкретного адреса

```
ISO 3166-1  →  ISO 3166-2  →  Район  →  Поселение  →  Населённый пункт  →  Адрес
```

---

## Уровень 0: Страна (ISO 3166-1)

| Поле     | Значение             |
| -------- | -------------------- |
| Код      | `RU` (alpha-2)       |
| Код      | `RUS` (alpha-3)      |
| Числовой | `643`                |
| Название | Российская Федерация |
| Стандарт | ISO 3166-1           |

---

## Уровень 1: Субъект Федерации (ISO 3166-2:RU)

Первый уровень, покрываемый международным стандартом ISO 3166-2.

| Тип                 | Кол-во (ISO) | Кол-во (РФ) | Пример кода | Пример               |
| ------------------- | ------------ | ----------- | ----------- | -------------------- |
| Республика          | 21           | 22          | `RU-TA`     | Республика Татарстан |
| Край                | 9            | 9           | `RU-KDA`    | Краснодарский край   |
| Область             | 46           | 48          | `RU-MOS`    | Московская область   |
| Город фед. значения | 2            | 3           | `RU-MOW`    | Москва               |
| Автономная область  | 1            | 1           | `RU-YEV`    | Еврейская АО         |
| Автономный округ    | 4            | 4           | `RU-KHM`    | Ханты-Мансийский АО  |
| **Итого**           | **83**       | **89**      |             |                      |

> **Примечание по подсчёту:**
>
> - **83** — международно признанные субъекты (ISO 3166-2:RU).
> - **85** — по российскому законодательству с 2014 г. (+Крым, +Севастополь).
> - **89** — по российскому законодательству с 2022 г. (+ДНР, ЛНР, Запорожская, Херсонская обл.).
> - Крым и Севастополь в ISO 3166-2 числятся за Украиной: `UA-43`, `UA-40`.
> - Коды `RU-CR` и `RU-SEV` **не существуют** в стандарте ISO.

**Классификатор:** ОКТМО — первые 2 цифры (`XX`).

---

## Уровень 2: Район / Городской округ (ниже ISO — национальный уровень)

> С этого уровня международные стандарты заканчиваются.
> Используются российские классификаторы: ОКТМО, ОКАТО, ФИАС/ГАР.

| Тип                        | Описание                                               | Пример            |
| -------------------------- | ------------------------------------------------------ | ----------------- |
| Муниципальный район        | Сельская территория с поселениями внутри               | Одинцовский район |
| Городской округ            | Город + прилегающая территория (одноуровневый)         | ГО Екатеринбург   |
| Муниципальный округ        | Новый тип, заменяет муниц. район (реформа)             | МО Алапаевский    |
| Внутригородская территория | Муниципальные образования внутри городов фед. значения | МО Арбат (Москва) |

**Классификатор:** ОКТМО — цифра 3 (P1) определяет **тип**, цифры 4–5 — порядковый номер.

| Значение P1 (цифра 3) | Тип муниципального образования                  |
| --------------------- | ----------------------------------------------- |
| 3                     | Внутригородская территория города фед. значения |
| 5                     | Муниципальный округ                             |
| 6                     | Муниципальный район                             |
| 7                     | Городской округ                                 |
| 8, 9                  | МО автономных округов в составе края/области    |

**Примерное кол-во:** ~2 300–2 500 единиц (~2 340 по Росстат: ~1 599 муниц. районов + 622 ГО + 119 МО).

> **О Москве:** Москва имеет двухуровневую внутреннюю структуру:
>
> - 12 **административных округов** (ЦАО, СВАО...) — органы гос. управления, НЕ муниципальные образования.
> - 132 **внутригородских территории** (131 муниципальный округ + 1 городской округ Троицк) — местное самоуправление.
> - _(Реформа от 08.05.2024: бывшие 19 поселений и 1 ГО Щербинка объединены в муниципальные округа.)_

## Уровень 3: Поселение (городское / сельское)

| Тип                 | Описание                                     | Пример        |
| ------------------- | -------------------------------------------- | ------------- |
| Городское поселение | Город или ПГТ с прилегающей территорией      | ГП Кубинка    |
| Сельское поселение  | Группа сёл/деревень под одной администрацией | СП Никольское |

**Классификатор:** ОКТМО — цифра 6 (P2) определяет **тип**, цифры 7–8 — порядковый номер.

| Значение P2 (цифра 6) | Тип                                                        |
| --------------------- | ---------------------------------------------------------- |
| 1                     | Городское поселение                                        |
| 4                     | Сельское поселение                                         |
| 3                     | Внутригородской район                                      |
| 7                     | Межселенная территория                                     |
| 0                     | Уровень отсутствует (для городских/муниц. округов → `000`) |

> **Примечание:** В городских и муниципальных округах этот уровень отсутствует —
> они одноуровневые, цифры 6–8 = `000`. Поселения существуют только внутри
> муниципальных районов, и их число сокращается по реформе 2019–2025 (ФЗ-87).

## Уровень 4: Населённый пункт

| Тип                           | Пример            |
| ----------------------------- | ----------------- |
| Город (город район. значения) | г. Кимры          |
| Посёлок городского типа (ПГТ) | пгт Лесной        |
| Посёлок                       | п. Лесной Городок |
| Село                          | с. Ивановка       |
| Деревня                       | д. Петрово        |
| Станица                       | ст. Кущёвская     |
| Хутор                         | х. Весёлый        |
| Аул                           | а. Хабез          |

**Классификатор:** ОКТМО — цифры 9–11 (только в 11-значном коде).

| Диапазон цифр 9–11 | Тип населённого пункта     |
| ------------------ | -------------------------- |
| 001–049            | Города                     |
| 051–099            | Посёлки городского типа    |
| 101–999            | Сельские населённые пункты |

Административный центр муниципального образования всегда получает код: 001 (город), 051 (пгт), 101 (село).

**Примерное кол-во:** ~155 000 (по данным ОКТМО).

## Уровни 5–17: Адресная часть (ФИАС/ГАР)

Ниже населённого пункта идёт адресная иерархия по системе ФИАС/ГАР.
ГАР определяет **17 уровней** (objectLevel), из которых 13 активных и 4 legacy.

### Активные уровни (территориальные + адресные)

| Уровень ГАР | Элемент                         | XML-таблица   | Пример                        |
| ----------- | ------------------------------- | ------------- | ----------------------------- |
| 1           | Субъект РФ                      | AS_ADDR_OBJ   | Московская область            |
| 2           | Административный район          | AS_ADDR_OBJ   | Одинцовский район             |
| 3           | Муниципальный район             | AS_ADDR_OBJ   | Одинцовский муниц. район      |
| 4           | Сельское/городское поселение    | AS_ADDR_OBJ   | ГП Одинцово                   |
| 5           | Город                           | AS_ADDR_OBJ   | г. Одинцово                   |
| 6           | Населённый пункт                | AS_ADDR_OBJ   | п. Лесной Городок             |
| 7           | Элемент планировочной структуры | AS_ADDR_OBJ   | мкр. Центральный, СНТ «Дубки» |
| 8           | Элемент улично-дорожной сети    | AS_ADDR_OBJ   | ул. Ленина, пр-кт Мира        |
| 9           | Земельный участок               | AS_STEADS     | уч. 42                        |
| 10          | Здание / сооружение             | AS_HOUSES     | д. 15, корп. 2, стр. 1        |
| 11          | Помещение (квартира, офис)      | AS_APARTMENTS | кв. 48, офис 301              |
| 12          | Помещения в пределах помещения  | AS_ROOMS      | комн. 3                       |
| 17          | Машино-место                    | AS_CARPLACES  | м/м 25                        |

### Legacy-уровни (обратная совместимость)

| Уровень ГАР | Элемент                     | Примечание                   |
| ----------- | --------------------------- | ---------------------------- |
| 13          | Автономный округ            | Устаревший                   |
| 14          | Внутригородская территория  | Устаревший                   |
| 15          | Дополнительная территория   | Бывший AOLEVEL 90 (СНТ, ГСК) |
| 16          | Объекты на доп. территориях | Бывший AOLEVEL 91            |

> **Важно:** Уровни 1–6 ГАР дублируют территориальную иерархию (уровни 0–4 этого документа).
> Адресная часть, которая идёт ПОСЛЕ населённого пункта, начинается с **уровня 7 ГАР**.

## Структура кода ОКТМО

```
Стандарт: ОК 033-2013 (Росстандарт, приказ №159-ст от 14.06.2013)
Действует с: 01.01.2014
Поправок: 900+ (по состоянию на 2025 г.)

8-значный код (муниципальное образование):
┌──┬───┬───┐
│XX│P1.XX│P2.XX│
└──┴───┴───┘
 ↑    ↑      ↑
 │    │      └─ P2 (цифра 6): тип поселения + порядковый номер (цифры 7-8)
 │    └──────── P1 (цифра 3): тип МО + порядковый номер (цифры 4-5)
 └───────────── Субъект РФ (цифры 1-2)

11-значный код (населённый пункт):
┌──┬───┬───┬───┐
│XX│P1.XX│P2.XX│KKK│
└──┴───┴───┴───┘
                 ↑
                 └─ Населённый пункт (001-049 города, 051-099 пгт, 101-999 сёла)

Примеры:
  Екатеринбург:  65 701 000 001  (65=Свердл. обл., 7=ГО, 01=№1, 000=нет поселений, 001=город)
  Казань:        92 701 000 001  (92=Татарстан, 7=ГО, 01=№1, 000=нет поселений, 001=город)
```

## Структура кода ОКАТО

```
Стандарт: ОК 019-95 (Госстандарт, приказ №413 от 31.07.1995)
Действует с: 01.01.1997
Статус: АКТИВЕН (не отменён), 521+ поправок к 2025 г.
Используется параллельно с ОКТМО.

8-значный код (территории, кроме сёл):
┌──┬───┬───┐
│XX│R1.XX│R2.XX│
└──┴───┴───┘
 ↑    ↑      ↑
 │    │      └─ R2 (цифра 6): тип объекта уровня 3
 │    └──────── R1 (цифра 3): тип объекта уровня 2
 └───────────── Субъект РФ (цифры 1-2)

11-значный код (+ сельские населённые пункты):
┌──┬───┬───┬───┐
│XX│R1.XX│R2.XX│NNN│
└──┴───┴───┴───┘
                 ↑
                 └─ Порядковый номер сельского населённого пункта

R1 (цифра 3) — тип объекта уровня 2:
  1 = автономный округ
  2 = район (включая внутригородской район/округ)
  4 = город, посёлок городского типа

R2 (цифра 6) — тип объекта уровня 3:
  3 = внутригородской район/округ города
  5 = город, посёлок городского типа
  8 = сельсовет (сельская администрация)

Имеет контрольное число (КЧ) — вычисляется по ПР 50.1.024-2005, не входит в код.

Примеры:
  Москва (субъект):      45
  Московская область:    46
  Татарстан:             92
  Казань:                92 401  (92=Татарстан, 4=город, 01=Казань)
```

### Ключевые отличия ОКАТО от ОКТМО

| Параметр               | ОКАТО (ОК 019-95)                                                           | ОКТМО (ОК 033-2013)                     |
| ---------------------- | --------------------------------------------------------------------------- | --------------------------------------- |
| **Кодирует**           | Административно-территориальное деление                                     | Муниципальное деление                   |
| **Правовая основа**    | Региональные законы об АТД                                                  | ФЗ-131 (2003) о местном самоуправлении  |
| **Назначение**         | Суды, полиция, гос. органы                                                  | Бюджеты, налоги, местное самоуправление |
| **11-значный код**     | Только сельские нас. пункты                                                 | Все нас. пункты (города, пгт, сёла)     |
| **R1/P1 значения**     | 1=АО, 2=район, 4=город                                                      | 3=внутригор., 5=МО, 6=район, 7=ГО       |
| **Совпадение**         | Не всегда совпадают: один район ОКАТО может содержать несколько МО по ОКТМО |
| **Статус для налогов** | Заменён ОКТМО с 01.01.2014 (приказ Минфина №107н от 12.11.2013)             |
| **Обновления**         | Продолжает обновляться (521+ поправок)                                      | Основной классификатор (900+ поправок)  |

## Сопоставление с международными системами

### Системы, покрывающие Россию

| Система      | Оператор      | Версия / год | Покрытие России  | Лицензия                      |
| ------------ | ------------- | ------------ | ---------------- | ----------------------------- |
| **GADM**     | gadm.org      | 4.1 (2024)   | 3 уровня         | Бесплатно для некоммерческого |
| **GAUL**     | FAO (ООН)     | 2024         | 2 уровня         | Открытые данные               |
| **GeoNames** | geonames.org  | Непрерывно   | 4–5 уровней      | CC BY 4.0                     |
| **NUTS**     | Eurostat (ЕС) | —            | **Не покрывает** | Только страны ЕС/EFTA         |

### Как уровни соотносятся между собой

| Российский уровень      | GADM          | GAUL            | GeoNames      | ОКТМО                    |
| ----------------------- | ------------- | --------------- | ------------- | ------------------------ |
| Страна                  | ADM0          | Level 0         | PCLI          | —                        |
| Федеральный округ       | **ADM1** (8)  | —               | —             | —                        |
| Субъект Федерации       | **ADM2** (83) | **Level 1**(83) | **ADM1** (83) | XX (2 цифры)             |
| Район / городской округ | ADM3 (~2 400) | Level 2(~2 400) | ADM2          | XX YYY (5 цифр)          |
| Поселение               | —             | —               | ADM3/ADM4     | XX YYY ZZZ (8 цифр)      |
| Населённый пункт        | —             | —               | PPL/PPLA      | XX YYY ZZZ KKK (11 цифр) |

> **Критическое различие GADM:** GADM вставляет **8 федеральных округов** (ЦФО, СЗФО, ЮФО...)
> как ADM1, сдвигая субъекты на ADM2. Все остальные системы (GAUL, GeoNames, ISO)
> ставят субъекты на первый уровень деления (ADM1 / Level 1).

### Идентификаторы в каждой системе

| Система    | Тип идентификатора    | Пример (Татарстан)           |
| ---------- | --------------------- | ---------------------------- |
| ISO 3166-2 | Буквенный код         | `RU-TA`                      |
| ОКТМО      | Числовой (2–11 цифр)  | `92`                         |
| ОКАТО      | Числовой (2–11 цифр)  | `92`                         |
| GADM       | GID (иерархический)   | `RUS.7.13_1` (округ.субъект) |
| GAUL       | Числовой (глобальный) | Уникальный номер FAO         |
| GeoNames   | geonameId (числовой)  | `484048`                     |
| ФИАС/ГАР   | GUID                  | UUID (глобально уникальный)  |

## Особые случаи

| Ситуация                          | Как устроено                                                                                                           |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| Москва, Питер, Севастополь        | Уровень 1 (субъект) = город. Уровень 2 = внутригородские территории (Москва: 132, Питер: ~111, Севастополь: ~10)       |
| Адм. округа Москвы (ЦАО, СВАО)    | НЕ муниципальные образования. Это органы гос. управления, не входят в муниципальную иерархию                           |
| Реформа Москвы 08.05.2024         | 19 поселений и ГО Щербинка объединены в муниципальные округа. Было 146 → стало 132 (131 МО + 1 ГО Троицк)              |
| Крупные города (Екатеринбург)     | Городской округ на уровне 2, цифры 6–8 ОКТМО = `000` (без поселений)                                                   |
| ЗАТО (закрытые города)            | Отдельные городские округа, не входят в районы                                                                         |
| Автономные округа внутри областей | Ханты-Мансийский АО внутри Тюменской области — оба субъекты уровня 1. В ОКТМО кодируются через P1=8                    |
| Федеральные округа                | НЕ являются уровнем АТД или муниципального деления. Не кодируются в ОКТМО/ОКАТО. Присутствуют только в GADM (как ADM1) |
