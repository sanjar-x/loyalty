Создание пользовательской сессии

Создает пользовательскую сессию для созданий партий и добавления заказов в партию.

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
Параметры запроса
Локальный URL /1.0/user-session
Тип POST
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
Тело запроса Нажмите на элемент структуры, чтобы увидеть описаниескопировать схему
Структура
{

{ - СимволНачало структуры

"batch-name": "string"

batch-name - Строка (Опционально)Номер партии

"session-type": "CREATE_BATCHES"

session-type - СтрокаТип пользовательской сессии = ['CREATE_BATCHES', 'ADD_BACKLOGS_TO_BATCH']

}

} - СимволОкончание структуры
