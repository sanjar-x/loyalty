Генерация печатной формы Ф112ЭК

Генерирует и возвращает pdf-файл с заполненной формой Ф112ЭК для указанного заказа. Только для заказа с «наложенным платежом». Если заказ не имеет данного атрибута, метод вернет ошибку. Если параметр sending-date не передается, берется текущая дата.

Введите токен авторизации приложения, ключ авторизации пользователя и другие параметры, тогда значения заголовков и параметров будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
id
Уникальный идентификатор заказа
sending-date (Опционально)
Дата отправки в почтовое отделение (yyyy-MM-dd)
Параметры запроса
Локальный URL /1.0/forms/{{ id }}{id}/f112pdf{{data.urlParamsString}}
Тип GET
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
