Закрытие пользовательской сессии для формирования партий и добавления отправлений

Закрывает пользовательскую сессию (добавление и удаление ШПИ заказов блокируется), запускает фоновый процесс формирования партий из заказов или добавления заказов в существующую партию.

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
Идентификатор пользовательской сессии
sending-date (Опционально)
Дата сдачи в почтовое отделение (yyyy-MM-dd)
timezone-offset (Опционально)
Смещение даты сдачи от UTC в секундах
use-online-balance
(Опционально)
Признак использования онлайн баланса
Параметры запроса
Локальный URL /1.0/user-session/{{ sessionUuid }}/close-session{{ buildUrlParams({ sendingDate: { value: sendingDate, prefix: 'sending-date' }, timezoneOffset: { value: timezoneOffset, prefix: 'timezone-offset' }, useOnlineBalance: { value: useOnlineBalance, prefix: 'use-online-balance' } }) }}
Тип POST
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
Ответ на запрос В успешном случае ответ 200 OK с пустым телом.
