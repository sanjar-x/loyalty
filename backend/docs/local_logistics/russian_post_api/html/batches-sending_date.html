Изменение дня отправки в почтовое отделение

Изменяет (устанавливает) новый день отправки в почтовое отделение.

Введите токен авторизации приложения, ключ авторизации пользователя и другие параметры, тогда значения заголовков и параметров будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
name
Наименование партии
year
Дата сдачи в почтовое отделение: год
month
Дата сдачи в почтовое отделение: месяц
dayOfMonth
Дата сдачи в почтовое отделение: день
Параметры запроса
Локальный URL /1.0/batch/{{ batchName }}{name}/sending/{{ year }}{year}/{{ month }}{month}/{{ dayOfMonth }}{dayOfMonth}
Тип POST
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
Ответ на запрос Нажмите на элемент структуры, чтобы увидеть описаниескопировать схему
