Поиск краткой информации об отделении ОПС по индексу

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
postal-code
Индекс почтового отделения
Параметры запроса
Локальный URL /postoffice/1.0/{{ postalCode }}{postal-code}/brief
Тип GET
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
