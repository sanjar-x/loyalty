Удаление отдельного возвратного отправления

Удаляет отдельное возвратное отправление.

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
barcode
ШПИ возвратного отправления
Параметры запроса
Локальный URL /1.0/returns/delete-separate-return?barcode={{ barcode }}
Тип DELETE
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
Ответ на запрос Нажмите на элемент структуры, чтобы увидеть описаниескопировать схему
