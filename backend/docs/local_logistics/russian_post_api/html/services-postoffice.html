Поиск почтового отделения по индексу

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
postal-code
Индекс почтового отделения
latitude
Широта
longitude
Долгота
current-date-time
Текущее время в формате yyyy-MM-dd'T'HH:mm:ss (Опционально)
filter-by-office-type
Фильтр по типам объектов в ответе. true: ГОПС, СОПС, ПОЧТОМАТ. false: все. Значение по-умолчанию - true.
ufps-postal-code
true: добавлять в ответ индекс УФПС для найдённого отделения, false: не добавлять. Значение по-умолчанию: false.
Параметры запроса
Локальный URL /postoffice/1.0/{{ postalCode }}{postal-code}{{ url }}
Тип GET
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
