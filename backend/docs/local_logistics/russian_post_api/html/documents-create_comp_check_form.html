Генерация печатной формы акта осмотра содержимого

Генерирует и возвращает pdf файл с формой акта осмотра содержимого для указанной партии.

Введите токен авторизации приложения, ключ авторизации пользователя и другие параметры, тогда значения заголовков и параметров будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
name
Наименование партии
Параметры запроса
Локальный URL /1.0/forms/{{ id }}{id}/completeness-checking-form
Тип GET
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
