# Запрос данных о партиях в архиве (долгосрочный)

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/long-term-archive/shipment/search?query={query}** |
| Тип | **GET** |

**Query-параметры:**

| Параметр | Описание |
|----------|----------|
| query | Условие для поиска: номер заказа или ШПИ |

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |
