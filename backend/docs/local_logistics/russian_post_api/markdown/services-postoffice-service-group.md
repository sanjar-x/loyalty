### Поиск почтовых сервисов ОПС по идентификатору группы сервисов

**Параметры запроса**

| Параметр | Описание |
|----------|----------|
| Локальный URL | `/postoffice/1.0/{postal-code}/services/{service-group-id}` |
| Тип | **GET** |

Параметры URL:

| Параметр | Тип | Описание |
|----------|-----|----------|
| **postal-code** | Строка | Индекс почтового отделения |
| **service-group-id** | Число | Идентификатор группы сервисов |

**Заголовки запроса**

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | AccessToken {token} |
| **X-User-Authorization** | Basic {key} |
| **Content-Type** | application/json;charset=UTF-8 |

**Ответ на запрос**

```json
[
  {
    "code": "string",
    "group-id": 0,
    "name": "string"
  }
]
```

| Поле | Тип | Описание |
|------|-----|----------|
| code | Строка | Код сервиса (например: 1.2) |
| group-id | Целое число | Внутренний идентификатор группы, которой принадлежит сервис |
| name | Строка | Описание сервиса |
