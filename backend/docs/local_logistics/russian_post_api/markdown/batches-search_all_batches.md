# Поиск всех партий

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/batch** |
| Тип | **GET** |

Query-параметры (все опционально):
- **mailType** -- Тип отправления
- **mailCategory** -- Категория отправления
- **size** -- Количество записей на странице
- **sort** -- Критерии сортировки: asc (по возрастанию) или desc (по убыванию). По умолчанию по возрастанию
- **page** -- Номер страницы (0..N)

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Ответ на запрос

Возвращает массив объектов партий. Структура каждого объекта партии описана в [Поиск партии по наименованию](batches-find_batch.md).
