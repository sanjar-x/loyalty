### Поиск населенного пункта по индексу

**Параметры запроса**

| Параметр | Описание |
|----------|----------|
| Локальный URL | `/postoffice/1.0/{postal-code}/settlement` |
| Тип | **GET** |

Параметры URL:

| Параметр | Тип | Описание |
|----------|-----|----------|
| **postal-code** | Строка | Индекс почтового отделения |

**Заголовки запроса**

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | AccessToken {token} |
| **X-User-Authorization** | Basic {key} |
| **Content-Type** | application/json;charset=UTF-8 |

**Ответ на запрос**

```json
{
  "postal-code": "string",
  "region": "string",
  "settlement": "string"
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| postal-code | Строка (Опционально) | Индекс почтового отделения |
| region | Строка (Опционально) | Область, край |
| settlement | Строка (Опционально) | Наименование населенного пункта |
