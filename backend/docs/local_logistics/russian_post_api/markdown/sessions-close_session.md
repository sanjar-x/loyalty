### Закрытие пользовательской сессии для формирования партий и добавления отправлений

Закрывает пользовательскую сессию (добавление и удаление ШПИ заказов блокируется), запускает фоновый процесс формирования партий из заказов или добавления заказов в существующую партию.

**Параметры запроса**

| Параметр | Описание |
|----------|----------|
| Локальный URL | `/1.0/user-session/{sessionUuid}/close-session` |
| Тип | **POST** |

Параметры URL:

| Параметр | Тип | Описание |
|----------|-----|----------|
| **sending-date** | Строка (Опционально) | Дата сдачи в почтовое отделение (yyyy-MM-dd) |
| **timezone-offset** | Число (Опционально) | Смещение даты сдачи от UTC в секундах |
| **use-online-balance** | Логический (Опционально) | Признак использования онлайн баланса |

**Заголовки запроса**

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | AccessToken {token} |
| **X-User-Authorization** | Basic {key} |
| **Content-Type** | application/json;charset=UTF-8 |

**Ответ на запрос**

В успешном случае ответ 200 OK с пустым телом.
