# Текущие точки сдачи

Возвращает список текущих точек сдачи.

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/user-shipping-points** |
| Тип | **GET** |

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Ответ на запрос

```json
[
  {
    "enabled": true,
    "operator-postcode": "string",
    "ops-address": "string"
  }
]
```

| Параметр | Тип | Описание |
|----------|-----|----------|
| enabled | Логическое (Опционально) | Признак активации |
| operator-postcode | Строка | Индекс почтового отделения обслуживания |
| ops-address | Строка (Опционально) | Адрес ОПС |
