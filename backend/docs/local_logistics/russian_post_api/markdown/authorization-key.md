# Ключ авторизации пользователя

**Ключ авторизации пользователя** генерируется с помощью алгоритма base64.
Перед кодированием имя и пароль, выданные сервисом passport.pochta.ru, разделяются двоеточием.

Пример: base64(login:password) = bG9naW46cGFzc3dvcmQ=

**Внимание! Не сохраняйте и не храните ваш пароль к API в текстовых файлах, так как это может привести к его хищению и/или компрометации.
Не оставляйте записанные на бумаге и т.п. носителях информации токен авторизации, ключ авторизации, логин и пароль к API, не передавайте эти данные третьим лицам!**

### Генерация ключа авторизации

Вы можете сгенерировать ключ авторизации здесь или можете воспользоватся сторонним сервисом.

### Добавление ключа в заголовок запроса

Добавьте параметр авторизации пользователя **X-User-Authorization** и укажите префикс **Basic**, символ "пробела" и **ключ авторизации пользователя**.

Пример:

| Наименование | Значение |
|---|---|
| X-User-Authorization | Basic bG9naW46cGFzc3dvcmQ= |
