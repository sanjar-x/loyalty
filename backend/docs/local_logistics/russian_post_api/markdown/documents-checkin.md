### Подготовка и отправка электронной формы Ф103

Присваивает уникальную версию партии для дальнейшего приема этой партии сотрудниками ОПС.
Отправляет по e-mail электронную форму Ф103 в ОПС для регистрации.

**Параметры запроса**

| Параметр | Описание |
|----------|----------|
| Локальный URL | `/1.0/batch/{name}/checkin` |
| Тип | **POST** |

Параметры URL:

| Параметр | Тип | Описание |
|----------|-----|----------|
| **name** | Строка | Наименование партии |
| **useOnlineBalance** | Логический (Опционально) | Признак использования онлайн баланса |

**Заголовки запроса**

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | AccessToken {token} |
| **X-User-Authorization** | Basic {key} |
| **Content-Type** | application/json;charset=UTF-8 |

**Ответ на запрос**

```json
{
  "error-code": "UNDEFINED",
  "f103-sent": true
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| error-code | Строка (Опционально) | Код ошибки |
| f103-sent | Логический (Опционально) | Отослана ли электронная версия ф103/ф103п в ОПС? |
