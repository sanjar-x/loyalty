### Удаление сессии

Удаляет пользовательскую сессию.

**Параметры запроса**

| Параметр | Описание |
|----------|----------|
| Локальный URL | `/1.0/user-session/{sessionUuid}` |
| Тип | **DELETE** |

**Заголовки запроса**

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | AccessToken {token} |
| **X-User-Authorization** | Basic {key} |
| **Content-Type** | application/json;charset=UTF-8 |

**Ответ на запрос**

В успешном случае ответ 200 OK с пустым телом.

В случае ошибки:

| Код ошибки | Описание |
|------------|----------|
| SESSION_IN_PROGRESS | Сессия в процессе выполнения |
