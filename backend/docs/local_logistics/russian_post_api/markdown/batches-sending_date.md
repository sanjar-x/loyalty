# Изменение дня отправки в почтовое отделение

Изменяет (устанавливает) новый день отправки в почтовое отделение.

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/batch/{name}/sending/{year}/{month}/{dayOfMonth}** |
| Тип | **POST** |

- **name** -- Наименование партии
- **year** -- Дата сдачи в почтовое отделение: год
- **month** -- Дата сдачи в почтовое отделение: месяц
- **dayOfMonth** -- Дата сдачи в почтовое отделение: день

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Ответ на запрос

```json
{
  "error-code": "UNDEFINED",
  "f103-sent": true
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| error-code | Строка (Опционально) | Код ошибки |
| f103-sent | Логические: true или false (Опционально) | Отослана ли электронная версия ф103/ф103п в ОПС? |
