# Редактирование отдельного возвратного отправления

Редактирование отдельного возвратного отправления (ЛВ).

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/returns/{barcode}** |
| Тип | **POST** |

**barcode** -- ШПИ возвратного отправления.

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Тело запроса

```json
{
  "address-from": { ... },
  "address-to": { ... },
  "insr-value": 0,
  "order-num": "string",
  "postoffice-code": "string",
  "recipient-name": "string",
  "sender-name": "string"
}
```

## Ответ на запрос

```json
{
  "errors": [
    {
      "code": "DIRECT_SHIPMENT_NOT_FOUND",
      "description": "string"
    }
  ]
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| errors | Массив | Ошибка при редактировании возвратного отправления |
| errors[].code | Строка | Код ошибки |
| errors[].description | Строка | Описание ошибки |
