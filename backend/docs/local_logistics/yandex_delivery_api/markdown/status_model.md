## Доставка «до двери»

![Схема статусной модели — доставка до двери](https://yandex.com/support/delivery-profile/docs-assets/support-delivery-profile/rev/r19043516/ru/_assets/status-model-door.svg)

### Описание статусов (до двери)

| Статус | Описание | Тип |
|--------|----------|-----|
| **`DRAFT`** | **Заказ создан** | Основной |
| `VALIDATING` | Заявка находится на проверке | Детализация |
| `VALIDATING_ERROR` | Заказ не подтвержден в сортировочном центре | Детализация |
| **`CREATED`** | **Заказ создан и подтвержден** | Основной |
| `DELIVERY_PROCESSING_STARTED` | Заказ создается в сортировочном центре | Детализация |
| `DELIVERY_TRACK_RECIEVED` | Заказ создан в системе службы доставки | Детализация |
| `SORTING_CENTER_PROCESSING_STARTED` | Заказ начал обрабатываться в сортировочном центре | Детализация |
| `SORTING_CENTER_TRACK_RECEIVED` | Заказ обработан в сортировочном центре | Детализация |
| `SORTING_CENTER_TRACK_LOADED` | Заказ создан в сортировочном центре | Детализация |
| `DELIVERY_LOADED` | Заказ добавлен в текущую отгрузку | Детализация |
| `SORTING_CENTER_LOADED` | Заказ подтвержден в сортировочном центре | Детализация |
| **`SORTING_CENTER_AT_START`** | **Заказ поступил в точку приема** | Основной |
| `SORTING_CENTER_PREPARED` | Заказ готов к отправке в службу доставки | Детализация |
| `SORTING_CENTER_TRANSMITTED` | Заказ доставляется до последней мили | Детализация |
| `DELIVERY_AT_START` | Заказ находится в городе получателя и готовится к отправке курьером | Детализация |
| `DELIVERY_AT_START_SORT` | Заказ находится в городе получателя и готовится к отправке курьером | Детализация |
| `DELIVERY_TRANSPORTATION_RECIPIENT` | Заказ доставляется клиенту | Детализация |
| `DELIVERY_TRANSMITTED_TO_RECIPIENT` | Заказ выдан получателю | Детализация |
| `DELIVERY_ATTEMPT_FAILED` | Неудачная попытка вручения заказа | Детализация |
| **`DELIVERY_DELIVERED`** | **Заказ вручен клиенту** | Основной |

### Статусы при отмене заказа (до двери)

| Статус | Описание | Тип |
|--------|----------|-----|
| **`CANCELLED`** | **Отмена** | Основной |

#### Возможные причины отмены

| Причина | Описание |
|---------|----------|
| `CANCELLED_BY_RECIPIENT` | Заказ отменен по просьбе клиента |
| `CANCELLED_USER` | Заказ отменен пользователем |
| `CANCELLED_IN_PLATFORM` | Заказ отменен в платформе |
| `SORTING_CENTER_CANCELLED` | Заказ отменен сортировочным центром |

### Статусы при возврате заказа (до двери)

| Статус | Описание | Тип |
|--------|----------|-----|
| `SORTING_CENTER_RETURN_PREPARING` | Заказ готовится к возврату | Детализация |
| `SORTING_CENTER_RETURN_PREPARING_SENDER` | Заказ готов к отправке отправителю | Детализация |
| `SORTING_CENTER_RETURN_ARRIVED` | Заказ доставлен в пункт отправителя | Детализация |
| `SORTING_CENTER_RETURN_RETURNED` | Заказ возвращен отправителю | Детализация |
| **`RETURN_PREPARING`** | **Заказ в процессе возврата на складе сортировочного центра** | Основной |
| **`RETURN_TRANSPORTATION_STARTED`** | **Заказ едет в точку выдачи** | Основной |
| **`RETURN_ARRIVED_DELIVERY`** | **Заказ возвращен на склад** | Основной |
| **`RETURN_TRANSMITTED_FULFILMENT`** | **Заказ передан на единый склад** | Основной |
| **`RETURN_READY_FOR_PICKUP`** | **Заказ готов для передачи магазину** | Основной |
| **`RETURN_RETURNED`** | **Заказ возвращен в магазин** | Основной |

### Статусы при переносе доставки

| Статус | Описание |
|--------|----------|
| `DELIVERY_UPDATED_BY_SHOP` | Доставка заказа перенесена отправителем |
| `DELIVERY_UPDATED_BY_RECIPIENT` | Доставка заказа перенесена получателем |
| `DELIVERY_UPDATED_BY_DELIVERY` | Доставка заказа перенесена службой доставки |
