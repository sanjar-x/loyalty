# Редактирование заказа

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/backlog/{id}** |
| Тип | **PUT** |

**id** -- Внутренний идентификатор отправления.

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Тело запроса

Тело запроса идентично телу запроса создания заказа (см. [Создание заказа](orders-creating_order.md)), но передаётся один объект (не массив).

## Ответ на запрос

```json
{
  "error-codes": [
    {
      "code": "UNDEFINED",
      "description": "string",
      "details": "string",
      "position": 0
    }
  ],
  "position": 0
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| error-codes | Массив | Список кодов ошибок |
| error-codes[].code | Строка | Код ошибки |
| error-codes[].description | Строка (Опционально) | Описание ошибки |
| error-codes[].details | Строка (Опционально) | Описание ошибки |
| error-codes[].position | Целое число (Опционально) | Индекс в массиве |
| position | Целое число (Опционально) | Индекс в массиве |
