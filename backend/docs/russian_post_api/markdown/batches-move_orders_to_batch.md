# Перенос заказов в партию

Переносит подготовленные заказы в указанную партию. Если часть заказов не может быть помещена в партию (тип и категория партии не соответствует типу и категории заказа) - возвращается json объект с указанием индекса заказа в переданном массиве и типом ошибки, остальные заказы помещаются в указанную партию. Каждому перенесенному заказу автоматически присваивается ШПИ.

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/batch/{name}/shipment** |
| Тип | **POST** |

**name** -- Наименование партии.

Query-параметр: **group-name** -- Идентификатор группы ММО (опционально).

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Тело запроса

```json
[
  0
]
```

Массив внутренних идентификаторов заказов.

## Ответ на запрос

```json
{
  "errors": [
    {
      "error-code": "UNDEFINED",
      "error-details": "string",
      "position": 0
    }
  ],
  "result-ids": [
    0
  ]
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| errors | Массив (Опционально) | Список ошибок |
| errors[].error-code | Строка (Опционально) | Код ошибки |
| errors[].error-details | Строка (Опционально) | Описание ошибки |
| errors[].position | Целое число (Опционально) | Индекс в исходном массиве |
| result-ids | Массив (Опционально) | Список успешно перенесенных в партию внутренних идентификаторов отправлений |
