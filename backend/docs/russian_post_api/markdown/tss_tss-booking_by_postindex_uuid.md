# Формирование заказа

Заменяет UUID заказа, созданный на этапе предварительного бронирования, на ШПИ.

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/external/v1/booking-by-postindex/{uuid}** |
| Тип | **PUT** |

**Параметры пути:**

| Параметр | Описание |
|----------|----------|
| uuid | UUID предварительного бронирования, будет заменен на реальный ШПИ из параметра barcode |

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Тело запроса

```json
{
  "barcode": "string",
  "contractNumber": "string"
}
```

| Параметр | Тип | Описание |
|----------|-----|----------|
| barcode | Строка | ШПИ отправления. Текстовая строка от 13 до 50 символов |
| contractNumber | Строка | Номер договора (максимальная длина 50 символов) |

## Ответ на запрос (ошибка)

```json
{
  "code": "string",
  "message": "string",
  "errors": [
    {
      "field": "string",
      "message": "string"
    }
  ]
}
```

| Параметр | Тип | Описание |
|----------|-----|----------|
| code | Строка | Текстовый идентификатор ошибки |
| message | Строка | Подробное описание ошибки поля входных данных |
| errors | Массив | Возвращается при наличии ошибок в полях входных данных |
| errors[].field | Строка | Текстовый идентификатор поля входных данных |
| errors[].message | Строка | Подробное описание ошибки поля входных данных |
