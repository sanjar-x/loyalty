# Создание возвратного отправления для ранее созданного отправления

Создает возвратное отправление (ЛВ) для уже созданного в ЛК отправления.

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/returns** |
| Тип | **PUT** |

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Тело запроса

```json
{
  "direct-barcode": "string",
  "mail-type": "UNDEFINED"
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| direct-barcode | Строка | ШПИ прямого отправления |
| mail-type | Строка | Вид РПО |

## Ответ на запрос

```json
{
  "errors": [
    {
      "code": "DIRECT_SHIPMENT_NOT_FOUND",
      "description": "string"
    }
  ],
  "return-barcode": "string"
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| errors | Массив | Ошибка создания возвратного отправления |
| errors[].code | Строка | Код ошибки |
| errors[].description | Строка | Описание ошибки |
| return-barcode | Строка | ШПИ возвратного отправления |
