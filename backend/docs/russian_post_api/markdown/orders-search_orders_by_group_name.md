# Поиск заказов по идентификатору группы

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/backlog/by-group-name/{group-name}** |
| Тип | **GET** |

**group-name** -- Идентификатор группы отправлений.

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Ответ на запрос

Возвращает массив объектов заказов. Структура каждого объекта заказа идентична телу запроса создания заказа (см. [Создание заказа](orders-creating_order.md)) с дополнительными полями, заполненными системой.
