# Нормализация ФИО

Очищает, разделяет и помещает значения ФИО в соответствующие поля возвращаемого объекта. Параметр id (идентификатор записи) используется для установления соответствия переданных и полученных записей, так как порядок сортировки возвращаемых записей не гарантируется.

**Параметры запроса**

| Параметр | Значение |
|---|---|
| Локальный URL | **/1.0/clean/physical** |
| Тип | **POST** |

**Заголовки запроса**

| Заголовок | Значение |
|---|---|
| **Authorization** | **AccessToken {токен}** |
| **X-User-Authorization** | **Basic {ключ}** |
| **Content-Type** | **application/json;charset=UTF-8** |

**Тело запроса**

```json
[
  {
    "id": "string",
    "original-fio": "string"
  }
]
```

| Поле | Тип | Описание |
|---|---|---|
| id | Строка | Идентификатор записи |
| original-fio | Строка | Оригинальные фамилия, имя, отчество одной строкой |

**Ответ на запрос**

```json
[
  {
    "id": "string",
    "middle-name": "string",
    "name": "string",
    "original-fio": "string",
    "quality-code": "CONFIRMED_MANUALLY",
    "surname": "string",
    "valid": true
  }
]
```

| Поле | Тип | Описание |
|---|---|---|
| id | Строка | Идентификатор записи |
| middle-name | Строка | Отчество |
| name | Строка | Имя |
| original-fio | Строка | Оригинальные фамилия, имя, отчество одной строкой |
| quality-code | Строка | [Код качества нормализации ФИО](#/enums-clean-fio-quality) |
| surname | Строка | Фамилия |
| valid | Логические: true или false | Приемлемое ли качество произведенной очистки? |
