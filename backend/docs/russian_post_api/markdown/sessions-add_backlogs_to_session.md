### Добавление заказов в пользовательскую сессию

Добавляет ШПИ заказов в пользовательскую сессию для дальнейшего формирования из них партий или добавления заказов в уже существующую партию.

**Параметры запроса**

| Параметр | Описание |
|----------|----------|
| Локальный URL | `/1.0/user-session/{sessionUuid}/add-backlogs-to-session` |
| Тип | **POST** |

**Заголовки запроса**

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | AccessToken {token} |
| **X-User-Authorization** | Basic {key} |
| **Content-Type** | application/json;charset=UTF-8 |

**Тело запроса**

```json
[
  "string"
]
```

Массив ШПИ — список ШПИ заказов.

**Ответ на запрос**

```json
[
  {
    "barcode": "string",
    "error": "ADDED_TO_SESSION"
  }
]
```

| Поле | Тип | Описание |
|------|-----|----------|
| barcode | Строка (Опционально) | ШПИ отправления |
| error | Строка | Код ошибки = ['ADDED_TO_SESSION'] |
