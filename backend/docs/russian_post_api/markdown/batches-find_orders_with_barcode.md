# Поиск заказов с ШПИ

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/shipment/search?query={query}** |
| Тип | **GET** |

**query** -- Строка поиска (ШПИ, номер заказа и т.д.).

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Ответ на запрос

Возвращает массив объектов заказов в партии. Структура каждого объекта заказа идентична телу запроса создания заказа (см. [Создание заказа](orders-creating_order.md)) с дополнительными полями, заполненными системой (включая ШПИ, статусы и т.д.).
