# Отмена заказа

Отменяет отправление, зарегистрированное на таймслот, по ШПИ или UUID.

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/external/v1/booking-by-postindex/{uuidOrBarcode}** |
| Тип | **DELETE** |

**Параметры пути:**

| Параметр | Описание |
|----------|----------|
| uuidOrBarcode | UUID предварительного бронирования или ШПИ |

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Ответ на запрос (ошибка)

```json
{
  "code": "string",
  "message": "string",
  "errors": [
    {
      "field": "string",
      "message": "string"
    }
  ]
}
```

| Параметр | Тип | Описание |
|----------|-----|----------|
| code | Строка | Текстовый идентификатор ошибки |
| message | Строка | Подробное описание ошибки поля входных данных |
| errors | Массив | Возвращается при наличии ошибок в полях входных данных |
| errors[].field | Строка | Текстовый идентификатор поля входных данных |
| errors[].message | Строка | Подробное описание ошибки поля входных данных |
