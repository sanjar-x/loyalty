# Поиск заказа по идентификатору

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/backlog/{id}** |
| Тип | **GET** |

**id** -- Внутренний идентификатор отправления.

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Ответ на запрос

Возвращает объект заказа. Структура объекта заказа идентична телу запроса создания заказа (см. [Создание заказа](orders-creating_order.md)) с дополнительными полями, заполненными системой.
