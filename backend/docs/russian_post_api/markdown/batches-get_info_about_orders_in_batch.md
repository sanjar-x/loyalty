# Запрос данных о заказах в партии

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/batch/{name}/shipment** |
| Тип | **GET** |

**name** -- Наименование партии.

Query-параметры (опционально):
- **size** -- Количество записей на странице
- **sort** -- Критерии сортировки: asc (по возрастанию) или desc (по убыванию). По умолчанию по возрастанию
- **page** -- Номер страницы (0..N)

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Ответ на запрос

Возвращает массив объектов заказов в партии. Структура каждого объекта заказа идентична телу запроса создания заказа (см. [Создание заказа](orders-creating_order.md)) с дополнительными полями, заполненными системой (включая ШПИ, статусы и т.д.).
