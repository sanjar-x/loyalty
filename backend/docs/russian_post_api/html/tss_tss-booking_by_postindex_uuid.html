Формирование заказа

Заменяет UUID заказа, созданный на этапе предварительного бронирования на ШПИ

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
uuid
будет заменен на реальный ШПИ из параметра barcode
Параметры запроса
Локальный URL /external/v1/booking-by-postindex/{{uuid}}
Тип PUT
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
Тело запроса Нажмите на элемент структуры, чтобы увидеть описаниескопировать схему

"barcode": "string"

contractNumber - СтрокаНомер договора (максимальная длина 50 символов)

}
