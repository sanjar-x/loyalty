Удаление сессии

Удаляет пользовательскую сессию.

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
Идентификатор пользовательской сессии
Параметры запроса
Локальный URL /1.0/user-session/{{ sessionUuid }}
Тип DELETE
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
Ответ на запрос В успешном случае ответ 200 OK с пустым телом. В случае ошибки (нажмите на элемент структуры, чтобы увидеть описание):
Структура
"SESSION_IN_PROGRESS"

Код ошибкиКод ошибки = ['SESSION_IN_PROGRESS']
