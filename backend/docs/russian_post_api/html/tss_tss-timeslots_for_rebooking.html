Получение списка свободных таймслотов при перебронировании

Получение списка свободных таймслотов при перебронировании. Данный метод необходимо вызывать, когда заказ существует в системе на любых этапах обработки.

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
uuidOrBarcode
UUID предварительного бронирования или ШПИ
contractNumber (Опционально)
Номер договора
plannedShippingDate
Плановая дата в формате: yyyy-MM-dd
Параметры запроса
Локальный URL /external/v1/timeslots-for-rebooking{{ buildUrlParams({ uuidOrBarcode: { value: uuidOrBarcode, prefix: 'uuidOrBarcode' }, contractNumber: { value: contractNumber, prefix: 'contractNumber' }, plannedShippingDate: { value: plannedShippingDate, prefix: 'plannedShippingDate' } }) }}
Тип GET
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
