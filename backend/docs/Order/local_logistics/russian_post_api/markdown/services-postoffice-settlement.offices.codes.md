### Поиск почтовых индексов в населённом пункте

**Параметры запроса**

| Параметр | Описание |
|----------|----------|
| Локальный URL | `/postoffice/1.0/settlement.offices.codes` |
| Тип | **GET** |

Параметры URL:

| Параметр | Тип | Описание |
|----------|-----|----------|
| **settlement** | Строка | Название населённого пункта (например Екатеринбург) |
| **region** | Строка | Область/край/республика, где расположен населённый пункт (например Свердловская) |
| **district** | Строка | Район, где расположен населённый пункт (для деревень, посёлков и т. д. — например Сухоложский) |

**Заголовки запроса**

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | AccessToken {token} |
| **X-User-Authorization** | Basic {key} |
| **Content-Type** | application/json;charset=UTF-8 |

**Ответ на запрос**

```json
["string"]
```

Массив почтовых индексов.
