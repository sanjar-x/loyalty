# Нормализация телефона

Принимает номера телефонов в неотформатированном виде, который может включать пробелы, символы: +-().
Очищает, разделяет и помещает сущности телефона (код города, номер) в соответствующие поля возвращаемого объекта.
Если номер телефона 11-ти значный (мобильный), то дополнительные параметры, кроме original-phone и id, указывать не обязательно.
Если номер телефона стационарный, то необходимо опционально указать дополнительные параметры для определения кода города.
Параметр id (идентификатор записи) используется для установления соответствия переданных и полученных записей, так как порядок сортировки возвращаемых записей не гарантируется.

**Параметры запроса**

| Параметр | Значение |
|---|---|
| Локальный URL | **/1.0/clean/phone** |
| Тип | **POST** |

**Заголовки запроса**

| Заголовок | Значение |
|---|---|
| **Authorization** | **AccessToken {токен}** |
| **X-User-Authorization** | **Basic {ключ}** |
| **Content-Type** | **application/json;charset=UTF-8** |

**Тело запроса**

```json
[
  {
    "area": "string",
    "id": "string",
    "original-phone": "string",
    "place": "string",
    "region": "string"
  }
]
```

| Поле | Тип | Описание |
|---|---|---|
| area | Строка (Опционально) | Область/край телефонного номера |
| id | Строка | Идентификатор записи |
| original-phone | Строка | Оригинальные номер одной строкой |
| place | Строка (Опционально) | Город телефонного номера |
| region | Строка (Опционально) | Регион телефонного номера |

**Ответ на запрос**

```json
[
  {
    "id": "string",
    "original-phone": "string",
    "phone-city-code": "string",
    "phone-country-code": "string",
    "phone-extension": "string",
    "phone-number": "string",
    "quality-code": "CONFIRMED_MANUALLY"
  }
]
```

| Поле | Тип | Описание |
|---|---|---|
| id | Строка | Идентификатор записи |
| original-phone | Строка | Оригинальные номер одной строкой |
| phone-city-code | Строка | Код города |
| phone-country-code | Строка | Код страны |
| phone-extension | Строка | Добавочный номер |
| phone-number | Строка | Телефонный номер |
| quality-code | Строка | [Код качества нормализации телефона](#/enums-clean-fio-phone-quality) |
