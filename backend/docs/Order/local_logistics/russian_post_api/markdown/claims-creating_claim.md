# Оформление заявления на дополнительную услугу

Создает заявление на предоставление дополнительной услуги к отправлению.

## Параметры запроса

| Параметр | Значение |
|----------|----------|
| Локальный URL | **/1.0/claims/createClaim** |
| Тип | **PUT** |

## Заголовки запроса

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | **AccessToken {token}** |
| **X-User-Authorization** | **Basic {key}** |
| **Content-Type** | **application/json;charset=UTF-8** |

## Тело запроса

```json
{
  "barcode": "string",
  "claimer": {
    "last-name": "string",
    "first-name": "string",
    "middle-name": "string"
  },
  "claim-type": "EARLY_RETURN",
  "org-name": "string",
  "org-inn": "string",
  "agreement-number": "string"
}
```

| Параметр | Тип | Описание |
|----------|-----|----------|
| barcode | Строка | Идентификатор отправления |
| claimer | Объект | Информация о заявителе |
| claimer.last-name | Строка | Фамилия заявителя |
| claimer.first-name | Строка | Имя заявителя |
| claimer.middle-name | Строка | Отчество заявителя |
| claim-type | Строка | Тип заявления. В настоящий момент для работы по API доступен только тип "EARLY_RETURN" -- "Досрочный возврат" |
| org-name | Строка | Наименование организации заявителя |
| org-inn | Строка | ИНН организации заявителя |
| agreement-number | Строка | Номер договора между организацией заявителя и Почтой России |

## Ответ на запрос

```json
{
  "code": "string",
  "sub-code": "string",
  "desc": "string"
}
```

| Параметр | Тип | Описание |
|----------|-----|----------|
| code | Строка | Код ошибки |
| sub-code | Строка (Опционально) | Дополнительный код ошибки |
| desc | Строка (Опционально) | Описание ошибки |
