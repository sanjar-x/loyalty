### Поиск обслуживающего ОПС по адресу

**Параметры запроса**

| Параметр | Описание |
|----------|----------|
| Локальный URL | `/postoffice/1.0/by-address` |
| Тип | **GET** |

Параметры URL:

| Параметр | Тип | Описание |
|----------|-----|----------|
| **address** | Строка | Строка с адресом. Следует учесть, что чем точнее адрес, тем точнее будет поиск. Пример: Санкт-Петербург, улица Победы, 15к1 |
| **top** | Число (Опционально) | Количество ближайших почтовых отделений в результате поиска. По умолчанию равно 3. |

**Заголовки запроса**

| Заголовок | Значение |
|-----------|----------|
| **Authorization** | AccessToken {token} |
| **X-User-Authorization** | Basic {key} |
| **Content-Type** | application/json;charset=UTF-8 |

**Ответ на запрос**

```json
{
  "is-matched": false,
  "postoffices": ["string"]
}
```

| Поле | Тип | Описание |
|------|-----|----------|
| is-matched | Логический | Флаг, говорящий, является ли переданный в параметре address адрес точным адресом ОПС (случай, когда переданный адрес и есть адрес ОПС). Данный флаг определяется только для первого ОПС в списке. |
| postoffices | Массив | Список почтовых индексов найденных ОПС отсортированный по релевантности |
