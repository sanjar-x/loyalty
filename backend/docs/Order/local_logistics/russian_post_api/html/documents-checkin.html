Подготовка и отправка электронной формы Ф103

Присваивает уникальную версию партии для дальнейшего приема этой партии сотрудниками ОПС. Отправляет по e-mail электронную форму Ф103 в ОПС для регистрации.

Введите токен авторизации приложения, ключ авторизации пользователя и другие параметры, тогда значения заголовков и параметров будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
name
Наименование партии
useOnlineBalance
(Опционально)
Признак использования онлайн баланса
Параметры запроса
Локальный URL /1.0/batch/{{ batchName }}{name}/checkin{{ buildUrlParams({ useOnlineBalance: { value: useOnlineBalance } }) }}
Тип POST
