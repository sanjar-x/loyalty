Текущее количество запросов по API

Возвращает общее количество запросов в сутки, разрешенное для пользователя, и остаток в текущих сутках.

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
Параметры запроса
Локальный URL /1.0/settings/limit
Тип GET
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
Ответ на запрос Нажмите на элемент структуры, чтобы увидеть описаниескопировать схему
