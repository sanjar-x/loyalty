Запрос данных о сессии

Запрашивает данные сессии (статус, состав для сессий в процессе, параметры для создания партий).

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
Идентификатор пользовательской сессии
Параметры запроса
Локальный URL /1.0/user-session/{{ sessionUuid }}
Тип GET
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
