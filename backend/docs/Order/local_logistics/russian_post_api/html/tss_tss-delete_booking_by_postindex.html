Отмена заказа

Отменяет отправление зарегистрированный на таймслот по ШПИ или UUID.

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
uuidOrBarcode
UUID предварительного бронирования или ШПИ.
Параметры запроса
Локальный URL /external/v1/booking-by-postindex/{{uuidOrBarcode}}
Тип DELETE
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
