Генерация возвратного ярлыка на одной печатной странице

Генерирует и возвращает pdf файл возвратного ярлыка на одной печатной странице


Введите токен авторизации приложения, ключ авторизации пользователя и другие параметры, тогда значения заголовков и параметров будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
barcode
ШПИ возвратного отправления
print-type (Опционально)
Тип печати
Параметры запроса
Локальный URL /1.0/forms/{{ barcode }}{barcode}/easy-return-pdf {{data.urlParamsString}}
Тип GET
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
