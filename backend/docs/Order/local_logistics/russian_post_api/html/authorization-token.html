Токен авторизации приложения

Токен авторизации приложения - это символьное обозначение вашего приложения.
API Онлайн-сервиса «Отправка» по токену отпределяет приложение, проводит первоначальную проверку и выделяет квоты на использование сервиса.

После активации доступа к API токен высылается на электронную почту, указанную при регистрации на онлайн-сервисе «Отправка» (otpravka.pochta.ru).
Также токен можно узнать в настройках личного кабинета.

Внимание! Не сохраняйте и не храните ваш пароль к API в текстовых файлах, так как это может привести к его хищению и/или компрометации.
Не оставляйте записанные на бумаге и т.п. носителях информации токен авторизации, ключ авторизации, логин и пароль к API, не передавайте эти данные третьим лицам!


Обновление токена

В настройках личного кабинета вы можете отправить запрос на обновление токена, нажав ссылку "Обновить токен". После завершения процесса обновления вы получите извещение об обновлении токена на электронную почту. Также будет изменен токен в настройках личного кабинета.

Добавление токена в заголовок запроса

Добавьте параметр авторизации приложения Authorization и укажите префикс AccessToken, символ "пробела" и токен авторизации приложения.

Пример:
Наименование Значение
Authorization AccessToken sDBaa9XNfFargSyQ8KIEM40GB_ndPmLu
