Получение списка свободных таймслотов

Получение списка свободных таймслотов с учетом плановой даты передачи груза на ОПС. Данный метод необходимо вызывать, когда заказ еще не создан.

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
postIndexFrom
Почтовый индекс отправителя
postIndexTo
Почтовый индекс получателя
contractNumber (Опционально)
Номер договора
plannedShippingDate
Плановая дата в формате: yyyy-MM-dd
address
Адрес получателя
workTypeCode
Код типа работ. В текущей реализации может принимать значение: delivery
mailTypeCode
Код типа отправления. В текущей реализации может принимать значение 24 (Курьер онлайн)
mailCtgCode
Код категории почтового отправления
Параметры запроса
Локальный URL /external/v1/timeslots-by-postindex{{ buildUrlParams({ postIndexFrom: { value: postIndexFrom, prefix: 'postIndexFrom' }, postIndexTo: { value: postIndexTo, prefix: 'postIndexTo' }, contractNumber: { value: contractNumber, prefix: 'contractNumber' }, plannedShippingDate: { value: plannedShippingDate, prefix: 'plannedShippingDate' }, address: { value: address, prefix: 'address' }, workTypeCode: { value: workTypeCode, prefix: 'workTypeCode' }, mailTypeCode: { value: mailTypeCode, prefix: 'mailTypeCode' }, mailCtgCode: { value: mailCategoryCode, prefix: 'mailCtgCode' } }) }}
Тип GET
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
