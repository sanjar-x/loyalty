Поиск почтовых сервисов ОПС по идентификатору группы сервисов

Введите токен авторизации приложения и ключ авторизации пользователя, тогда значения заголовков будут сгенерированы автоматически.

Токен авторизации
Ключ авторизации
postal-code
Индекс почтового отделения
service-group-id
Идентификатор группы сервисов.
Параметры запроса
Локальный URL /postoffice/1.0/{{ postalCode }}{postal-code}/services/{{ serviceGroupId }}{service-group-id}
Тип GET
Заголовки запроса
Authorization AccessToken {{ token }}
X-User-Authorization Basic {{ key }}
Content-Type application/json;charset=UTF-8
