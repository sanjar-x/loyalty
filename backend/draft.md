Основная концепция
Пользователь оформляет заказ через платформу и выбирает локальный пункт выдачи заказов (ПВЗ) для получения товара. Платформа выступает посредником между покупателем, зарубежными маркетплейсами и партнёрскими поставщиками.

После оформления заказа товар выкупается на соответствующей платформе и доставляется через логистическую компанию, затем передаётся в локальные ПВЗ для выдачи клиенту.
