# Правила работы с терминалом и тестами

При запуске тестов или выполнении команд в терминале СТРОГО соблюдай следующий порядок:

1. Сначала всегда активируй виртуальное окружение командой:
   `& c:/Users/Sanjar/Desktop/project/.venv/Scripts/Activate.ps1`
2. НИКОГДА не запускай тесты просто командой `pytest`.
3. Всегда используй менеджер пакетов `uv` для запуска тестов: `uv run pytest`.
