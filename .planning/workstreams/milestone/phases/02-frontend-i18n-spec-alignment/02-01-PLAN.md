From frontend/admin/src/lib/utils.js (line 48-51) — existing read-side i18n helper:
```javascript
export function i18n(obj, fallback = '') {
  if (!obj || typeof obj !== 'object') return fallback;
  return obj.ru ?? obj.en ?? Object.values(obj)[0] ?? fallback;
}
```

- CategoryCreateRequest: `name_i18n: I18nDict` (required) — serialized as `nameI18N`

```javascript
export function buildI18nPayload(ru, en) {
```
