- `test_rejects_parent_not_found`: Command with nonexistent parent_id. Assert `CategoryNotFoundError`, `uow.committed is False`.
- `test_rejects_template_not_found`: Command with nonexistent template_id. Assert `AttributeTemplateNotFoundError`, `uow.committed is False`.

- `test_template_id_ellipsis_means_no_change`: Update with default template_id (Ellipsis). Assert template propagation NOT triggered.
- `test_template_id_none_clears_template`: Pre-seed category with template_id. Update with `template_id=None`. Assert category's template_id is now None.
- `test_emits_category_updated_event`: Assert `CategoryUpdatedEvent` in `uow.collected_events`.
