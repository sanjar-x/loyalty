  Each test must first create a brand and category (prerequisites) using conftest helpers. A shared helper `create_product` should be defined at module level:
  ```python
  async def create_product(client, brand_id, category_id, *, slug=None, title_i18n=None):
      slug = slug or f"prod-{uuid4().hex[:8]}"
      title_i18n = title_i18n or {"ru": "Товар", "en": "Product"}
      payload = {"titleI18n": title_i18n, "slug": slug, "brandId": str(brand_id), "primaryCategoryId": str(category_id)}
      resp = await client.post("/api/v1/catalog/products", json=payload)
      assert resp.status_code == 201, resp.text
      return resp.json()
  ```

  Tests to write:
  - test_create_product_success: POST -> 201, response has `id`, `defaultVariantId`, `message`

  IMPORTANT: For attribute assignment, the attribute must:
  - Be level="product" and is_dictionary=true
  - Be bound to the template that's assigned to the product's category
  - Have a value to assign

  Setup flow per test:
  1. Create attribute template
  2. Create attribute (level="product", is_dictionary=true)
  3. Bind attribute to template
  4. Create category with template_id
  5. Create brand
  6. Create product with that brand + category
  7. Create attribute value
  8. Now assign attribute value to product
