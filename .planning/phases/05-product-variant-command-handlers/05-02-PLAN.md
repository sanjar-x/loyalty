**TestAddVariant class** (~4 test methods):
- `test_happy_path_without_price`: Pre-seed product in FakeUoW.products, call handler with name_i18n -> variant added, committed, result.variant_id is not None, product now has 2 variants (default + new)
- `test_happy_path_with_price`: command.default_price_amount=5000, default_price_currency="RUB" -> variant created with Money(5000, "RUB")
- `test_product_not_found`: Unknown product_id -> ProductNotFoundError, not committed
- `test_variant_added_to_product_aggregate`: After add, verify product.variants contains the new variant by checking result.variant_id appears in [v.id for v in product.variants]

**TestUpdateVariant class** (~5 test methods):

- `test_happy_path_update_price`: _provided_fields includes "default_price_amount", amount=3000, currency="USD" -> Money(3000, "USD") applied
- `test_product_not_found`: Unknown product_id -> ProductNotFoundError, not committed
- `test_variant_not_found`: Valid product but unknown variant_id -> VariantNotFoundError, not committed
- `test_invalid_currency_format`: _provided_fields includes "default_price_currency", value="ab" (not 3 uppercase ASCII) -> ValueError, not committed

**TestDeleteVariant class** (~4 test methods):
