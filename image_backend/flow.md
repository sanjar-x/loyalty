# Media Upload Flow — System Architecture

## Участники

```
┌───────────┐     ┌───────────┐     ┌───────────────┐     ┌───────────┐
│ Frontend  │     │  Backend  │     │ ImageBackend  │     │  Bucket   │
│ (Next.js) │     │ (бизнес-  │     │ (обработка +  │     │   (S3)    │
│           │     │  логика)  │     │  хранение     │     │           │
│           │     │           │     │  изображений) │     │           │
└───────────┘     └───────────┘     └───────────────┘     └───────────┘
```

---

## Зоны ответственности

| Участник         | Ответственность                                                 |
| ---------------- | --------------------------------------------------------------- |
| **Frontend**     | UI, local preview, вызов API, polling статуса                   |
| **Backend**      | Бизнес-логика: продукты, бренды, каталог. Хранит ссылку         |
| **ImageBackend** | Presigned URL, приём/обработка изображений, варианты, статус    |
| **Bucket**       | Хранение оригиналов и обработанных вариантов, direct upload/CDN |

---

     │  POST /api/v1/media/{mediaId}/confirm      │                      │

     │                     │                      │                      │

     │                     │                      │  ◄── 200 OK ──────── │

     │        url: "https://cdn.../thumb.jpg",    │                      │

     │      { role: "webp",                       │                      │

     │    ]                                       │                      │
     │  }                                         │                      │
     │                     │                      │                      │
     │  ┌──────────────────┐                      │                      │

     │                     │                      │                      │
